# SenseMeter
Official page for SenseMeter app, including app information.
SenseMeter is a free utility app for daily sensor data monitoring speed, altitude, acceleration, steps, noise, compass, etc.

# Privacy Policy

SenseMeter respects your privacy.
This app does not collect, store, or transmit any personal data.

Data Usage
-	All sensor data (such as motion, location, microphone input) is processed locally on your device only.
- No data is uploaded to any server.
-	No third-party analytics, advertising, or tracking services are used.
-	No login or account is required to use the app.

Permissions Explanation
SenseMeter may request access to the following system permissions only to provide core functionality:
-	Location: Used to display speed, maps, altitude, and compass information.
-	Motion: Used to measure acceleration, movement, and step count.
-	Microphone: Used to measure ambient noise levels.

These permissions are optional and can be managed in your device system settings at any time.
